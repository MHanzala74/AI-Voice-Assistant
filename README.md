# AI Voice Assistant

This project is a simple AI Voice Assistant built with Python and Streamlit. It listens to voice commands through the microphone and responds using text-to-speech.

## Features

* Play songs on YouTube
* Tell the current date and time
* Search information from Wikipedia
* Read saved phone numbers
* Start and stop the assistant using Streamlit buttons
* Stop listening using the **ESC** key

## Technologies Used

* Python
* Streamlit
* SpeechRecognition
* pyttsx3
* Wikipedia
* pywhatkit

## Run the Application

```bash
streamlit run main.py
```

After launching the app, click **Start Listening** and speak your command. Use the **Stop** button or press **ESC** to stop the assistant.
